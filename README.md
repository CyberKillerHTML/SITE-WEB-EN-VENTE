# SITE-WEB-EN-VENTE
Je vends un site WEB entièrement personnalisable et simple, adapté à n'importe quelle entreprise.

Merci pour votre achat !

Pour mettre votre site en ligne :
1. Connectez-vous à votre hébergement (OVH, IONOS, o2switch…)
2. Ouvrez le gestionnaire de fichiers
3. Glissez-déposez tous les fichiers du ZIP dans le dossier /www/
4. Votre site sera visible immédiatement

Pour toute modification, contactez-moi.
